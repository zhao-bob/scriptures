# 创世纪 39

## 约瑟和波提乏的妻子

1 同时，约瑟已被向下带到了埃及，在那里一个名叫波提乏的埃及人，法老的一个官员和护卫长，从将他带到那里的以实玛利人手中买了他。 2 而耶和华与约瑟同在，而他就成了一个成功的人，在他埃及主人的家室中服侍。

3 当他的主人看见耶和华与他同在并使他在他做的所有中兴旺时， 4 约瑟在他视域中得青睐并成为他的私人侍从。

波提乏让他主管他的家室并将他拥有的一切都托付给他。 5 从他让约瑟主管他的家室和他拥有的所有的那时起，耶和华为了他而祝福这个埃及人的家室。耶和华的祝福在他拥有的一切上，不管是在他的房子中还是在他的田地中。 6 所以波提乏将他拥有的所有都留给约瑟护理；除了他吃的食物，他自己一切都不操心。

现在约瑟体格健美而英俊， 7 而一些时间后他主人的妻子向约瑟抛媚眼并说，“与我同寝。”
