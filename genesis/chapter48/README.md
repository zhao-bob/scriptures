# 创世纪 48

## 雅各祝福以法莲和玛拿西

1 一段时间后约瑟被告知，“你的父亲病了。”所以他带着他两个儿子，玛拿西和以法莲出发。 2 当雅各被告知，“你的儿子已来到你这里，”以色列振作他的力量并在床上坐起。

3 雅各对约瑟说，“上帝全能者在迦南之的路斯向我显现，而在那里祂祝福我 4 并告诉我，‘看啊，我会使你富有果实且使你倍增；我会使你成为众多民族，并会将此地赐给你之后你的后代作为永久的财产。’

5 而现在，我到你这里之前，在埃及为你而生的你的两个儿子要被计数为我的；以法莲和玛拿西应是我的，正如流便和西缅是我的一样。 6 在他们之后，任何为你生的子女应是你的，而他们在他们继承的领地上应以他们哥哥的名被称呼。
