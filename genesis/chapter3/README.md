# 创世纪 3
genesis 3

## 大蛇的欺骗

The Serpent’s Deception

1 现在那大蛇比耶和华上帝已造的田野的任何走兽都狡黠。而他对那女人说，“上帝真的说，‘你决不能以园子中的任何树为食？’”

2 那女人回答那大蛇，“我们可以吃园子中树的果实， 3 但对于在园子中央的树的果实，上帝已说，‘你决不能吃它或摸它，否则你会死。’”

4 “你不一定会死，”那大蛇告诉那女人。“因为上帝知道在你吃它的那日，你的双眼会被打开而你会像上帝一样，知道善与恶。”

5 当那女人看见那树宜为食物且悦目，而这是为获得智慧所渴望的，她就取了果实并吃了它。她也将一些给了与她一同的丈夫，而他也吃了它。

6 他们俩的眼睛就都被打开了，而他们知道了他们是裸露的；所以他们把无花果叶缝在一起，而为他们自己做遮盖。
