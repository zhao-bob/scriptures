# 创世纪 49

## 雅各祝福祂的儿子们

1 然后雅各将他的儿子们唤来并说，“聚拢过来以使我能告诉你们在要来的日子中会发生在你们身上的事：

2 雅各的儿子们啊，到一起来并听；听你父亲以色列的话。

3 流便啊，你是我的长子，我的权能，是我力量的开端，尊贵出众，权柄出众。 4 但你像诸水一样不受控制，你就不会再出众，因为你走到了你父亲的床上，到了我的塌上，并玷污了它。

5 西缅和利未是兄弟；他们的刀剑是暴力的武器。 6 我愿永不进入他们的议会；我愿永不加入他们的集会。因为他们在他们的愤怒中杀人，一时兴起就砍断牛的腿筋。 7 他们的愤怒是该被咒诅的，因为它是强烈的！而他们的忿怒是该被咒诅的，因为它是冷酷的！我会使他们散布在雅各家中，并使他们分散在以色列之中。
