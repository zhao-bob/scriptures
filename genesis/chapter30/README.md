# 创世纪 30

## 但和拿弗他利

1 当拉结看见她不为雅各生任何孩子，她忌妒她的姐姐。“给我孩子，否则我会死！”她对雅各说。 

2 雅各对拉结变得愤怒并说，“我是在上帝的位置上吗？是祂已使你不生孩子。”

3 然后她说，“这是我的女仆辟拉。与她同寝，使她能为我生孩子，以使我借着她也能建立一个家庭。”

4 所以拉结将她的仆人辟拉给雅各做妻子，而他与她同寝， 5 辟拉就受孕并为他生了一个儿子。 6 然后拉结说，“上帝已洗清我；祂已听见我的恳请而赐给我一个儿子。”所以她给他取名但。

7 而拉结的仆人辟拉再次受孕并为雅各生了第二个儿子。 8 然后拉结说，“在我巨大的挣扎中，我已与我的姐姐角力并取胜。”所以她给他取名拿弗他利。
