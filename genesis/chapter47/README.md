# 创世纪 47

## 雅各在歌珊安家

1 所以约瑟去告诉法老：“我的父亲和我的兄弟们，与他们的羊群和牛群以及他们拥有的所有一起，已从迦南之地而来而现在正在歌珊。” 

2 而他选择了他兄弟中的五位并将他们呈在法老面前。

3 “你们的职业是什么？”法老问约瑟的兄弟们。

“你的仆人是牧羊人，”他们回复，“我们和我们的先祖都是。”

4 然后他们对法老说，“我们来到这地暂时居住，因为迦南之地的饥荒已很严重，在那里你的仆人们的羊群没有牧场。所以现在，请允许你的仆人们在歌珊之地安家。”

5 法老对约瑟说，“既然你的父亲和兄弟们已来到你这里， 6 埃及之地就在你面前；将你的父亲和兄弟们在此地最好的部分安家。他们能栖居在歌珊之地。而如果你了解他们之中有任何有才干的人，就使他们主管我自己的牲畜。”
