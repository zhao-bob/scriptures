# 创世纪 45

## 约瑟揭示他的身份

1 然后约瑟在他所有侍从面前再也不能控制他自己，而他大喊，“叫每个人都离开我出去！”

所以当他将他自己向他的兄弟们揭晓时，他们中没有一个与约瑟在一起。

2 但他哭泣得如此大声以致埃及人听见了他，而法老的家室不久就听闻了此事。

3 约瑟对他的兄弟们说，“我是约瑟！我的父亲仍然活着吗？”

但他们无能回答他，因为他们在他的面前惊恐。

4 “我是约瑟，你们的兄弟，”他说，“你们卖进埃及的那个！ 5 而现在，不要为你们将我卖入这地方而对你们自己忧痛或愤怒，因为是为了要拯救生命，上帝才在你们之前差派我。 6 因为这两年饥荒已笼罩这土地，而会再有五年没有耕种也没有收获。 7 上帝在你们之前差派我以保守你们作为地上的余民，并要由一次大解救来拯救你们生命。 8 因此，不是你们送我到这里，而是上帝，祂已立我作法老之父---他所有家室的主以及埃及所有土地之上的统治者。
